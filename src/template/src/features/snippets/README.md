# Snippets Folder

This folder is included mainly as a demo for Quarto include behavior.
If you do not plan to use reusable snippet includes, you usually do not need this folder.

Use Quarto include shortcodes to pull these files into a page:

```markdown
{{< include /features/snippets/quarto-shared-note.qmd >}}
```

If you do use snippets, keep shared notes, repeated callouts, and small content fragments here so you can maintain them in one place.
